# CTF Helper Toolkit

A small command-line toolkit to automate common recon steps for TryHackMe and Hack The Box machines (nmap scans, simple HTTP enumeration, basic wordlist mangling).

## scan

Run a standard nmap scan against a target IP and save the output.

(Later) http-enum: Try common paths and log status codes.

(Later) mangle: Simple wordlist mangler for passwords.

Python CLI first; C++ for performance-heavy components later.
